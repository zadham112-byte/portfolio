## Bug Title
Login button allows empty password

## Environment
- OS: Windows 10
- Browser: Chrome
- Version: 120

## Steps to Reproduce
1. Open login page
2. Enter valid username
3. Leave password field empty
4. Click Login

## Expected Result
User should not be able to login and error message should appear

## Actual Result
User logged in successfully without password

## Severity
High

## Priority
High

## Status
Open
